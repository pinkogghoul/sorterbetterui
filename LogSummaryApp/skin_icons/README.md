# Icon Directory

This directory should contain icon files for the application.

## Required Icons

- `vbucks_icon.png` - Icon for VBucks categories (30x30 pixels or larger)
- `skin_range_icon.png` - Icon for skin range categories (30x30 pixels or larger)

## Category Icons

Individual category icons should be named using sanitized category names:
- Convert to lowercase
- Replace spaces with underscores
- Remove special characters
- Add `.png` extension

Examples:
- "Skull Trooper" → `skull_trooper.png`
- "Merry Mint Axe" → `merry_mint_axe.png`
- "Black Knight" → `black_knight.png`

If an icon file is missing, a white star placeholder will be displayed.
